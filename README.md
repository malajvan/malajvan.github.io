# malajvan.github.io

### Install and run 

```shell
export NODE_OPTIONS=--openssl-legacy-provider
yarn
yarn start
```
### Update
Open `src/portfolio.js` and make changes.

### Build and deploy 

```shell
yarn build
yarn deploy
```
